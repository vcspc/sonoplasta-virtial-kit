# sonopalsta-virtual-kit

Este é o **Sonopalsta Virtual Kit**, uma solução modular para sonoplastia e foley virtual, focada em alto desempenho e código limpo.

## 📜 Constituição do Projeto

Este projeto é regido por uma [Constituição](./.specify/memory/constitution.md) que define nossos princípios fundamentais:

1.  **Clean Code & Modularidade**: Código legível e SOLID.
2.  **Documentação Proativa**: Documentação obrigatória na pasta `docs`.
3.  **Testes como Requisito**: TDD não negociável.
4.  **Performance e Baixa Latência**: Foco em processamento ágil.
5.  **Versionamento Semântico**: SemVer rigoroso.

## 🚀 Começando

A documentação detalhada pode ser encontrada na pasta [docs/](./docs/).

### Pré-requisitos

- Node.js (v18+)
- TypeScript
